Encode text

A function that accepts text selects all the unique words in it, calculates the number of occurrences for each word and assigns a unique code. Then encodes the text, replacing the words from the dictionary with the word code. (Word - symbols between spaces. Unique code - random number (preferably in 16 or 36 characters system))

  • Example: 
    Lorem ipsum ipsum dolor.
  • Result: 
    {
  dictionary: [
    { word: 'Lorem', count: 1, code: '58' },
    { word: 'ipsum', count: 2, code: 'rd' },
    { word: 'dolor.', count: 1, code: 'g7' }
  ],
  encodedText: '58,rd,rd,g7'
}
