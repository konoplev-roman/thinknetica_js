Text structure

A function that accepts text and returns an array of objects with a structure describing the words and the sum of codes of all characters of the word. (Word - symbols between spaces)

  • Example: 
    Lorem ipsum dolor sit amet.
  • Result: 
    [
  { word: 'Lorem', sum: 511 },
  { word: 'ipsum', sum: 558 },
  { word: 'dolor', sum: 544 },
  { word: 'sit', sum: 336 },
  { word: 'amet.', sum: 469 }
]
